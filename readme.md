# KISSCMS: Foursquare

Foursquare API bindings for KISSCMS


## Dependencies

* [KISSCMS](https://kisscms.com) > 2.0.0


## Install

Using git
```
$ git submodule add git://github.com/kisscms/foursquare.git ./plugins/foursquare
```
